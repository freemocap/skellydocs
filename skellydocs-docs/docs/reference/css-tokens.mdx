# CSS design tokens

skellydocs defines all its visual properties as CSS custom properties. These tokens are set in `custom.css` and used by every component.

## Full token list

### Backgrounds

| Token | Default | Description |
|---|---|---|
| `--sk-bg-deep` | `#06050e` | Main page background |
| `--sk-bg-surface` | `#0e0c1a` | Card and panel backgrounds |

### Borders

| Token | Default | Description |
|---|---|---|
| `--sk-border` | `#1a1730` | Default border color |

### Text

| Token | Default | Description |
|---|---|---|
| `--sk-text` | `#e8e6f0` | Primary text color |
| `--sk-text-dim` | `#8a87a0` | Muted/secondary text |

### Accents

| Token | Default | Description |
|---|---|---|
| `--sk-accent` | `#6ee7b7` | Primary accent (buttons, highlights) |
| `--sk-accent-dim` | `rgba(110,231,183,0.15)` | Accent tint for backgrounds |
| `--sk-purple` | `#a78bfa` | Secondary accent (links, badges) |

### Typography

| Token | Default | Description |
|---|---|---|
| `--sk-mono` | `JetBrains Mono, monospace` | Monospace font stack |

## Overriding tokens

Add overrides in your site's CSS file:

```css
:root {
  --sk-accent: #60a5fa;       /* blue accent */
  --sk-accent-dim: rgba(96, 165, 250, 0.15);
}
```

All components will pick up the new values automatically.
